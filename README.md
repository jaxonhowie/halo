# Halo

macOS 桌面像素风布偶猫宠物。纯 Swift + SpriteKit 实现，无第三方依赖。

## 功能

- **待机** — 猫咪坐在桌面，自然眨眼动画（12 帧，6fps）
- **走动** — 水平移动，碰壁自动转向（9 帧）
- **睡觉** — 蜷缩身体，漂浮 ZZZ 动画（4 帧）
- **想吃鱼** — 猫咪渴望吃鱼的可爱动画（11 帧）
- **跳跃** — 单击触发，带 "喵~" 文字气泡
- **抚摸** — 双击触发，显示漂浮爱心 + "咕噜咕噜~" 气泡（不打断当前状态）
- **拖拽移动** — 按住猫咪拖到桌面任意位置
- **右键菜单** — 手动切换状态、查看统计或退出
- **桌面置顶** — 透明无边框窗口，浮动在所有窗口之上
- **跨桌面** — 所有桌面空间可见
- **不在 Dock 显示** — 以 accessory 模式运行
- **窗口位置记忆** — 退出时自动保存位置，下次启动恢复
- **喝水提醒** — 监控键盘输入，工作 15 分钟后弹窗提醒喝水，支持倒计时气泡和每日统计
- **走动提醒** — 工作 60 分钟后弹窗提醒起身走动，独立倒计时气泡
- **工作时长** — 自动追踪连续工作时间，猫咪头顶显示计时，10 分钟无操作自动暂停，退出时自动保存
- **统计面板** — 右键打开，表格 + 柱状图展示今日喝水、走动、工作时长（echarts 渲染）

## 安装与启动

```bash
# 编译运行（一键）
./run.sh

# 或者分步执行
swift build
.build/debug/Halo

# Release 编译（优化）
swift build -c release

# 打包为 .app
./build_app.sh
open .build/Halo.app
```

要求 macOS 13+，Xcode Command Line Tools。

**喝水/走动提醒功能**需要辅助功能权限：系统设置 > 隐私与安全性 > 辅助功能。

## 基本操作

| 操作 | 效果 |
|------|------|
| 左键单击 | 猫咪跳跃 + "喵~" |
| 左键双击 | 爱心漂浮 + "咕噜咕噜~"（不打断当前状态） |
| 按住拖拽 | 移动猫咪位置 |
| 右键 | 打开菜单（待机/走动/睡觉/想吃鱼/查看统计/退出） |

猫咪会在待机、走动、睡觉之间自动切换状态。单击会打断当前状态触发跳跃，双击则不影响当前状态。

## 喝水/走动提醒

- 开始打字后自动启动倒计时（喝水 15 分钟，走动 60 分钟）
- 画布顶部显示带描边和阴影的倒计时气泡
- 倒计时结束后猫咪移到屏幕中央，弹出确认框
- 确认后猫咪回到原位，今日次数 +1

## 统计面板

右键菜单点击"查看统计"打开浮动面板：

- 表格展示：💧 喝水次数、🚶 走动次数、💻 工作时长
- 柱状图：echarts 渲染，直观对比今日各项数据
- 面板中打字不会触发喝水/走动提醒
- 数据每秒自动刷新

## 工作时长

- 首次按键自动开始计时，猫咪头顶显示 "💻 45m" 或 "💻 1:23"
- 10 分钟无键盘操作自动暂停，累计时间保存到本地
- 退出应用时自动保存当前会话
- 每日零点自动重置

## 项目结构

```
Sources/Halo/
├── main.swift              # AppKit 入口，透明无边框窗口、事件处理、提醒、统计面板
├── CatSpriteScene.swift    # SpriteKit 场景：状态机、动画、交互、倒计时气泡
├── SpriteData.swift        # Sprite 帧数据与颜色表（字符映射编码）
└── Resources/              # PNG 像素动画资源
    ├── idle/               # 待机动画（idle0-idle11，12 帧）
    ├── walk/               # 走动动画（walk0-walk8，9 帧）
    ├── sleep/              # 睡觉动画（sleep0-sleep3，4 帧）
    └── wantFish/           # 想吃鱼动画（wantFish0-wantFish10，11 帧）
```

## 技术实现

- **渲染** — SpriteKit，PNG 纹理以 `.nearest` 过滤保持像素锐利
- **窗口** — AppKit NSWindow，透明无边框，`.floating` 置顶，208x208 像素
- **状态机** — idle / walk / sleep / wantFish / jump，带权重随机切换
- **动画系统** — 固定时间步长（1/60s），各状态独立帧率
- **交互** — 单击跳跃，双击抚摸，拖拽移动，右键菜单
- **喝水/走动提醒** — `NSEvent.addGlobalMonitorForEvents` 监听键盘，`Timer` 计时，`NSAlert` 弹窗，`UserDefaults` 持久化
- **工作时长** — 键盘活动追踪，空闲 10 分钟自动暂停，退出时保存
- **统计面板** — `NSPanel` + `WKWebView`，内嵌 echarts 柱状图，深色主题
- **窗口位置记忆** — `UserDefaults` 存储坐标，启动时校验屏幕边界后恢复
- **倒计时气泡** — 多层 `SKLabelNode` 叠加实现描边（8 方向黑色偏移）+ 阴影（右下灰色偏移）
